# Estrutura do projeto
A estrutura de pastas do projeto é organizada da seguinte maneira:
```bash
    furia-fangirl/
    ├── backend/                      → código Flask
    │   ├── data/
    │   │   └── databases/ ...
    │   ├── services/ ...
    │   ├── app.py
    │   ├── Procfile
    │   └── requirements.txt
    ├── documentation/ ...            → documentação
    ├── frontend/                     → código React
    │   ├── public/ ...
    │   ├── src/
    │   │   ├── components/ ...
    │   │   ├── config/ ...
    │   │   ├── hooks/ ...
    │   │   ├── styles/ ...
    │   │   ├── types/ ...
    │   │   ├── App.tsx
    │   │   └── index.tsx
    │   ├── package.json
    │   └── tsconfig.json
    ├── screenshots/ ...
    ├── .env.example
    ├── .gitignore
    └── README.md
```
